  • Opa's Birthday Website

    A special website for my grandfather's birthday

    Learn More 

    I specially created this website for my grandfather's 73rd birthday. It has a number guessing game at the start, and once completed, there will be three virtual gifts containing videos of me.

    I created this website using HTML, CSS and JavaScript. For this website, the styling (CSS) and the scripting (JS) was pretty challenging. Nevertheless, I managed to complete it.

    How To Use

    • Type one digit in the text box. That digit will be your guess for the first digit of a number (tens place) between 25 to 50, eg. 3
    • If you guessed the number correctly, it will appear as the first digit of the number. Otherwise, it will appear at the bottom section under 'Wrong Digits'
    • Once both digits are guessed correctly, there will be a popup. Click 'Ok' to proceed
    • You will then be redirected to a page containing three gifts. Click any of the gifts to unbox it.
    • To go back and unbox another gift, click on the 'Back' button

  • Number Guessing Game

    A console-based number guessing game that is made using Python

    Learn More 

    While learning Python, I wanted to try to put my skills that I have learnt into use. So, I created this simple console-based number guessing game.

    Since this project was created using Python, it is hosted on a different website. You may follow the instructions given to find out how to play it.

    Note: For mobile users, press the "return" or "enter" key after keying in your input so that it can be registered in the game

    How To Use

    • Once you visited the website, click on the green button that states "Run"
    • You will be given a set of languages to choose from. It is recommended for you to choose English as translations for other languages may not be accurate.
    • Enter the language name or its number stated in the language options
    • You have to enter the first value in the range. For example, if you want to guess a number between 1 and 100, the first number of the range should be 1
    • Next, you will have to enter the second value in the range. For the example above, the second value should be 100
    • You're all set, have fun playing!
